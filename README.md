# SuperTicTacToe

## How to play?
Visit: ```https://www.youtube.com/shorts/_Na3a1ZrX7c```

## Clone The Project
```
git clone https://github.com/KhushPatibandha/superTicTacToe.git
cd superTicTacToe/
```

## Setup virtual env
```
python -m venv \venv
source venv/bin/activate
```

## Install Dependencies
```
pip3 install -r requirements.txt
```

## Console
```
python3 main.py
```

## GUI
```
python3 gui.py
```
